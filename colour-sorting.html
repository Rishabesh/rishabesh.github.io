Colour Sorting System

Projects

Designed and built a colour-sorting machine using a TCS230 sensor, OLED display and servo motor.

Features

  • Automatic calibration to lighting conditions
  • Token detection using RGB sensor
  • Real-time display on OLED screen
  • Sorting into pots via SG90 servo
  • Custom PCB shield + CAD mechanical parts
